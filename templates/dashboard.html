🧠 How the Seating Algorithm Works

1Branches are auto-split into 2 groups — Group A goes to odd columns, Group B goes to even columns
2Students from each group are lined up branch by branch in order
3Seats are filled room by room, row by row, column by column
4Odd columns → Group A students  |  Even columns → Group B students
5When one branch finishes → next branch in that group continues automatically
6No same-branch students sit beside or behind each other! ✅

💡 About This System

This Automatic Seating Arrangement System generates exam seating plans based on the number of rooms, branches, and student PIN numbers. Teachers can upload student keylists as CSV or Excel files, and the system automatically assigns seats ensuring no two students from the same branch sit adjacent to each other.
